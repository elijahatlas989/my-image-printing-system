# Project Report Outline

The supplied project specification asks for the following documentation:

1. Certificate of Completion
2. Table of Contents
3. Problem Definition
4. Customer Requirement Specification
5. Project Plan
6. E-R Diagram
7. Algorithms
8. GUI Standards Document
9. Interface Design Document
10. Task Sheet
11. Project Review and Monitoring Report
12. Unit Testing Check List
13. Final Check List

The original status report and feedback form templates are included in the `Documentation/Templates` folder.
